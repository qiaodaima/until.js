## 目录结构说明
```
utils
├── README.md
└── src
    ├── base.js    // 一些基础的、没有办法归类的操作
    ├── url.js     // 和url有关的操作，例如获取url参数、获取鼠标坐标
    ├── number.js  // 和数字有关的操作，例如价钱格式化、获取某个范围的随机数
    ├── regExp.js  // 常用正则表达式集合
    ├── rx.js      // 节流、防抖
    ├── screen.js  // 和屏幕有关的，例如判断一个元素是否在屏幕内
    └── time.js    // 和时间有关的操作，例如把时间戳转换为时间
```

> 使用方式请认真看源码，源码是最好的文档，不要辜负那字字句句中蕴含的深情~
